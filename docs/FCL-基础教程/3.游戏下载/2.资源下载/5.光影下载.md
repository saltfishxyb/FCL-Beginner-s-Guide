# 光影下载
光影的下载安装和模组类似，但是也有一点不同

# 光影加载器

## 光影加载器的选择
首先，我们需要明确需求
- 如果你只是想要开个光源，发一张截图，那么推荐**Fabric+Iris(性能更强)**或**原版客户端+OptiFine(更加简单)**
- 如果你想要在玩模组时加光影，推荐**NeoForge+Iris**或**Forge+OptiFine**
- 如果你注重性能，想要在玩原版时开光影，推荐**Fabric+Iris**

## 光影加载器的下载&安装
- OptiFine:

> 在下载版本时，点击OptiFine，即可在安装版本时将OptiFine一并安装

- Forge/NeoForge/Fabric+Embeddedium/Iris

> 因为Embeddedium和Iris都是模组，所以直接参考[模组安装](docs/FCL-基础教程/3.游戏下载/2.资源下载/2.模组下载.md)

# 光影的下载

## FCL内置下载
1. 打开资源下载页面，点击第一行第六个“**光影**”
![1.1.1.打开资源下载点击光影](/img/docs/FCL%20基础教程/3.游戏下载/2.资源下载/5.光影下载/1.1.1.打开资源下载点击光影.jpg)

2. 选择光影并点击打开到版本选择页面，选择对应的**游戏版本**
![1.1.2.打开版本选择页面](/img/docs/FCL%20基础教程/3.游戏下载/2.资源下载/5.光影下载/1.1.2.打开版本选择页面.jpg)

:::note
FCL会根据你的游戏版本自动置顶合适的版本，如没有特殊情况，选择置顶的版本即可
:::

3. 选择光影的版本并点击，一般情况下，最新即可
![1.1.3.选择光影版本](/img/docs/FCL%20基础教程/3.游戏下载/2.资源下载/5.光影下载/1.1.3.选择光影版本.jpg)
4. 点击确定开始下载，等待下载结束
![1.1.4.点击确定开始下载](/img/docs/FCL%20基础教程/3.游戏下载/2.资源下载/5.光影下载/1.1.4.点击确定开始下载.jpg)


## 浏览器下载
1. 在MT管理器**左侧窗口**找到在浏览器下载的光影包，通常在`/storage/emulated/0/Download/`。或使用浏览器的分享功能分享到MT管理器“**定位所在位置**”亦可
![1.2.1.打开光影](/img/docs/FCL%20基础教程/3.游戏下载/2.资源下载/5.光影下载/1.2.1.打开光影.jpg)

2. 在MT管理器**右侧窗口**操作，长按下方“**↑**”
![1.2.2.长按上箭头](/img/docs/FCL%20基础教程/3.游戏下载/2.资源下载/5.光影下载/1.2.2.长按上箭头.jpg)
![1.2.2.跳转路径窗口](/img/docs/FCL%20基础教程/3.游戏下载/2.资源下载/5.光影下载/1.2.2.跳转路径窗口.jpg)

3. 输入并跳转到`/storage/emulated/0/FCL/.minecraft/versions/[版本名称]/shaderpacks/`
![1.2.3.长按光影包](/img/docs/FCL%20基础教程/3.游戏下载/2.资源下载/5.光影下载/1.2.3.长按光影包.jpg)

4. 长按**左侧窗口**的光影包，点击**移动**，点击确定
![1.2.4.点击移动](/img/docs/FCL%20基础教程/3.游戏下载/2.资源下载/5.光影下载/1.2.4.点击移动.jpg)
![1.2.4.确定](/img/docs/FCL%20基础教程/3.游戏下载/2.资源下载/5.光影下载/1.2.4.确定.jpg)

:::note
即使在该路径中不存在`shaderpacks`文件夹，移动时MT也会自动帮你创建，所以不用担心跳转到了一个不存在的目录
:::
![1.2.最终展示](/img/docs/FCL%20基础教程/3.游戏下载/2.资源下载/5.光影下载/1.2.最终展示.jpg)

# 光影的加载
至此，你已经完成了光影的下载和安装，下面是光影的加载

## 原版/Forge+OptiFine：
1. 启动Minecraft，创建并打开世界

2. 按**ESC**，打开菜单，点击打开**选项**
![2.1.2.打开菜单点击选项](/img/docs/FCL%20基础教程/3.游戏下载/2.资源下载/5.光影下载/2.1.2.打开菜单点击选项.jpg)

3. 打开**视频设置**，点击**光影包**
![2.1.3.打开选项点击视频设置](/img/docs/FCL%20基础教程/3.游戏下载/2.资源下载/5.光影下载/2.1.3.打开选项点击视频设置.jpg)

4. 在光影包列表中单击需要加载的光影，等待加载
![2.1.4.点击光影](/img/docs/FCL%20基础教程/3.游戏下载/2.资源下载/5.光影下载/2.1.4.点击光影.jpg)

:::note
在光影包列表中，点击**光影设置**可以打开光影的配置界面
![2.1.5.光影设置](/img/docs/FCL%20基础教程/3.游戏下载/2.资源下载/5.光影下载/2.1.5.光影设置.jpg)
:::


## Fabric/NeoForge+Iris
1. 启动Minecraft，创建并打开世界

2. 按**ESC**，打开菜单，点击打开**选项**
![2.2.2.打开菜单点击选项](/img/docs/FCL%20基础教程/3.游戏下载/2.资源下载/5.光影下载/2.2.2.打开菜单点击选项.jpg)

3. 打开**视频设置**，点击**光影包**
![2.2.3.打开选项点击视频设置](/img/docs/FCL%20基础教程/3.游戏下载/2.资源下载/5.光影下载/2.2.3.打开选项点击视频设置.jpg)
![2.2.4.打开视频设置点击光影包](/img/docs/FCL%20基础教程/3.游戏下载/2.资源下载/5.光影下载/2.2.4.打开视频设置点击光影包.jpg)

4. 在光影包列表中单击需要加载的光影，点击**应用光影包**，等待加载
![2.2.5.点击光影包](/img/docs/FCL%20基础教程/3.游戏下载/2.资源下载/5.光影下载/2.2.5.点击光影包.jpg)
