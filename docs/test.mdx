## 跳转按钮（Docusaurus 原生）

Docusaurus 文档页面底部**自动生成**上一篇/下一篇导航按钮，无需手动添加。

如果你想在文档内部添加导航卡片，可以使用 DocCardList：

```jsx
<DocCardList />
```
或者手动指定卡片：
```
<DocCard item={{
  type: 'link',
  label: '上一篇：账号登录',
  docId: 'account-login',
  href: '/docs/account-login'
}} />
```
