# 什么是 Minecraft？

## Minecraft 是什么？

**Minecraft**（中文叫「**我的世界**」）是一款**沙盒游戏**（Sandbox Game），由瑞典程序员 **Markus "Notch" Persson** 在 2009 年开发，后来成立了 **Mojang** 工作室，2014 年被微软以 25 亿美元收购。

「沙盒游戏」是什么意思？打个比方：传统游戏像「**游乐园**」，有过山车、有迷宫、有剧情，你按设计好的路线玩。沙盒游戏像「**一盒积木**」，给你一堆方块，你想搭什么搭什么，没有固定目标，**自己定玩法**。

Minecraft 的核心玩法：

- **挖矿**：用镐子挖地下的矿石（煤、铁、金、钻石）。
- **建造**：用方块搭房子、城堡、城市。
- **合成**：用工作台把原料合成工具、武器、食物。
- **生存**：白天打怪、晚上躲怪、种田、养殖。
- **红石**：用红石电路做自动化机器（类似电路）。
- **探索**：去不同的维度（下界、末地）冒险。

## Minecraft 的画面

Minecraft 的画面是**方块风格**——所有东西都是方块组成的：方块的土地、方块的树、方块的人、方块的动物。这种风格叫「**体素图形**」（Voxel Graphics），和传统的多边形 3D 不同。

虽然画面看起来「简陋」，但**这种风格是 Minecraft 的灵魂**：

- 简单 → 任何电脑都能跑。
- 方块 → 玩家能自由搭建。
- 风格化 → 不追求真实，反而独特。

## Minecraft 的版本

Minecraft 有好几个版本，**最容易混淆**：

### 1. Java 版（Java Edition）

- **原版**，Notch 2009 年用 Java 写的。
- 运行在 **Windows、Mac、Linux** 电脑上。
- **需要装 Java**（JDK 17 或 21）。
- **mod 生态最丰富**，玩家最爱。
- 代码开源（反编译容易），所以 mod 多。
- 占内存大、启动慢（Java 的特性）。

### 2. 基岩版（Bedrock Edition）

- 微软 2011 年用 **C++** 重写的版本。
- 运行在 **Windows 10/11、手机、Xbox、PlayStation、Switch** 等多平台。
- **不需要 Java**。
- 性能好、启动快、跨平台联机。
- mod 生态弱（叫「Add-On」和「行为包」，不如 Java 版 mod 强大）。

### 3. 中国版

- 网易代理的版本，分 Java 版和基岩版。
- 内置 mod 商城、社交功能。
- 但**有广告、有氪金、mod 受限**，老玩家不爱用。

### 4. 教育版（Education Edition）

- 基岩版的衍生版，用于学校教学。

## Java 版和基岩版的对比

| 对比项 | Java 版 | 基岩版 |
|--------|---------|--------|
| 开发语言 | Java | C++ |
| 平台 | 主要是 Win/Mac/Linux 电脑 | Win10及以上系统/手机/主机 |
| 需要 Java | 是 | 否 |
| 性能 | 占内存、启动慢 | 性能好、启动快 |
| mod | **超多、超强大** | 少、叫 Add-On |
| 联机 | 电脑之间 | **跨平台**（手机+电脑+主机） |
| 价格 | ￥89（Java版+基岩版） | ￥89（Java版+基岩版） |
| 玩家偏好 | mod 玩家、老玩家 | 跨平台联机玩家 |

> 由于 Java版 本身的定位就是电脑端，所以这边Java联机没写**电脑+手机**，实际上 **手机Java版 与 电脑Java版** 是可以联机的

**简单说**：

- 想玩 mod、玩红石、玩老服务器 → **Java 版**。
- 想跨设备联机（手机+电脑+主机）、性能好 → **基岩版**。

## 为什么 Minecraft 这么火？

Minecraft 是**史上销量最高的游戏**（超过 3 亿份），原因：

### 1. 自由度极高

没有固定目标，你想怎么玩怎么玩。可以盖房子、挖矿、做机器、玩 RPG、玩 PvP，**玩法无限**。

### 2. 创造性

玩家搭出了**故宫、霍格沃茨、中土世界**等惊人作品。Minecraft 成了**数字版乐高**。

### 3. mod 生态

Java 版有海量 mod：工业 mod、魔法 mod、生物 mod、维度 mod，**几乎能改成任何游戏**。

### 4. 多人联机

服务器（Hypixel 等）有几十种小游戏，**像玩 MMO**。

### 5. 教育价值

被用于编程教学、建筑设计、历史还原。

### 6. 跨平台

基岩版能跨设备联机，覆盖所有玩家。

## Minecraft 和手机的关系

Minecraft 基岩版**原生支持手机**（安卓、iOS），从应用商店装就行。

但 **Java 版不能直接在手机上跑**，因为：

1. Java 版需要 JVM，手机没有官方 JVM。
2. Java 版用电脑的图形 API（OpenGL），手机用 OpenGL ES / Vulkan。
3. Java 版的代码是给电脑架构（x86）编译的，手机是 ARM 架构。

**怎么在手机上玩 Java 版？** 用 **Fold Craft Launcher** 等启动器，把电脑 Java 版「移植」到手机。具体原理下一篇讲。

## 小结

- **Minecraft = 2009 年 Notch 开发的沙盒游戏**，方块风格，自由度极高。
- 主流版本：**Java 版**（电脑，mod 多）和**基岩版**（多平台，性能好）。
- Java 版要装 Java，基岩版不要。
- **史上销量最高**，靠自由度、mod、联机、教育价值。
