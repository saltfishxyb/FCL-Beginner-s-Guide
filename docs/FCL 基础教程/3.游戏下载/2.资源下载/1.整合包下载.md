# 整合包下载
>FCL内置了整合包下载的功能，可以直接从CurseForge、Modrinth下载安装整合包。
FCL也可以从本地导入整合包

## 怎么下载&安装整合包
这个部分分为`内置下载、本地导入和互联网下载`


### 内置下载

#### 什么是内置下载
内置下载是指通在FCL内置的下载功能，从CurseForge和Modrinth下载整合包

#### 怎么用内置下载

1. 打开整合包下载页面。打开下载页面，点击第一行的第二个“**整合包**”
![打开整合包下载页面](/img/docs/FCL%20基础教程/3.游戏下载/2.资源下载/1.整合包下载/打开整合包下载页面.jpg)

2. 筛选和搜索你想要的整合包


:::note 筛选的方法
整合包列表左侧有筛选和搜索功能，筛选
:::

3. 点击你想要下载的整合包，打开版本列表页面
![1.3.打开版本列表页面](/img/docs/FCL%20基础教程/3.游戏下载/2.资源下载/1.整合包下载/1.3.打开版本列表页面.jpg)

4. 选择你想要的**游戏版本**，点击打开
![1.4.选择游戏版本](/img/docs/FCL%20基础教程/3.游戏下载/2.资源下载/1.整合包下载/1.4.选择游戏版本.jpg)

5. 选择你要安装的**整合包版本   **，点击开始下载
![1.5.选择整合包版本](/img/docs/FCL%20基础教程/3.游戏下载/2.资源下载/1.整合包下载/1.5.选择整合包版本.jpg)


:::note 不要混淆*游戏版本*和*整合包版本*
游戏版本是指**Minecraft的版本**，而整合包版本是**整合包的**
:::

6. 等待下载结束后，自动跳转整合包详情页面，点击**安装**，确认风险，开始安装
![安装](/img/docs/FCL%20基础教程/3.游戏下载/2.资源下载/1.整合包下载/安装.jpg)
![确认风险](../../../../static/img/docs/FCL%20基础教程/3.游戏下载/2.资源下载/1.整合包下载/确认风险.jpg)


:::danger 警告
- 如果你确定该整合包不会在手机上运行崩溃，可以直接无视风险。
- 如果你不确定，建议先看完风险提示。因为整合包一般不会适配手机，所以请不要想整合包作者提出手机端的崩溃问题，除非你明确了这不是手机端的问题
:::

1. 等待安装结束后，点击确认以完成安装
![确认风险](/img/docs/FCL%20基础教程/3.游戏下载/2.资源下载/1.整合包下载/确认风险.jpg)




### 本地导入

#### 什么是本地导入
本地导入是指从设备储存内导入已经下载下来的整合包

#### 怎么本地导入
1. 确定你想导入的整合包**所在的路径**。这里我们以` /storage/emulated/0/Download/ `下面有一个` Fabulously Optimized-5.10.2.mrpack `为例

2. 打开整合包下载页面。打开下载页面，点击第一行的第二个“**整合包**”
![打开整合包下载页面](/img/docs/FCL%20基础教程/3.游戏下载/2.资源下载/1.整合包下载/打开整合包下载页面.jpg)

3. 打开整合包安装页面。点击左侧下方的“**安装整合包**”
![2.2.点击安装整合包](/img/docs/FCL%20基础教程/3.游戏下载/2.资源下载/1.整合包下载/2.2.点击安装整合包.jpg)

4. 点击“**导入本地整合包文件 →**”，打开整合包路径选择页面
![2.4.打开路径选择页面](/img/docs/FCL%20基础教程/3.游戏下载/2.资源下载/1.整合包下载/2.4.打开路径选择页面.jpg)

5. 选择整合包路径。根据之前确定的整合包路径，选择文件夹（如：Download）
![2.5.选择路径](/img/docs/FCL%20基础教程/3.游戏下载/2.资源下载/1.整合包下载/2.5.选择路径.jpg)

6. 选择整合包。找到路径之后点击整合包文件（如` Fabulously Optimized-5.10.2.mrpack `），点击右侧的确定跳转回FCL整合包详情页面
![2.6.选择整合包](/img/docs/FCL%20基础教程/3.游戏下载/2.资源下载/1.整合包下载/2.6.选择整合包.jpg)

7. 点击**安装**，确认风险，开始安装
![确认风险](/img/docs/FCL%20基础教程/3.游戏下载/2.资源下载/1.整合包下载/确认风险.jpg)

:::danger 警告
- 如果你确定该整合包不会在手机上运行崩溃，可以直接无视风险。
- 如果你不确定，建议先看完风险提示。因为整合包一般不会适配手机，所以请不要向整合包作者提出**手机端的崩溃问题**，除非你确定了这不是手机端的问题。
:::

8. 等待安装结束后，点击完成以完成安装
![确认风险](/img/docs/FCL%20基础教程/3.游戏下载/2.资源下载/1.整合包下载/确认风险.jpg)
![确认安装完成](/img/docs/FCL%20基础教程/3.游戏下载/2.资源下载/1.整合包下载/确认安装完成.jpg)





### 互联网下载

> 讲真的这个功能也没什么用，无非就是比本地导入少手动下载和选择路径，方便一点而已。

> 很多时候整合包作者都会把整合包存在网盘，而国内的网盘基本都要客户端才能下载内容

> 所以在这里就不放图了(doge

#### 什么是互联网下载
互联网下载是指由启动器直接从互联网下载并导入整合包

这个功能和PCL的下载功能差不多，但是不能下载整合包以外的内容，并且下载完之后会自动跳转到整合包详情页面

#### 怎么互联网下载
1. 确定你想导入的整合包**连接**。这里我们以`https://github.com/Fabulously-Optimized/fabulously-optimized/releases/download/v13.2.2/Fabulously.Optimized-v13.2.2.mrpack`为例
2. 打开整合包下载页面。打开下载页面，点击第一行的第二个“**整合包**”
3. 打开整合包安装页面。点击左侧下方的“**安装整合包**”
4. 点击“**从互联网下载整合包 →**”，弹出“要下载的整合包链接”窗口
5. 输入整合包的链接，点击确认开始下载整合包
6. 等待下载完成后，自动跳转到整合包详情页面
7. 点击**安装**，确认风险，开始安装
8. 等待安装结束后，点击完成以完成安装
